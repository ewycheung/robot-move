# robot-move

## Description
Hello! Robot coming online.

## Command the robot with

### L - turn left
> Output: Robot at (x, y) facing Direction

### R - turn right
> Output: Robot at (x, y) facing Direction

### M - move forward
> Output: Robot at (x, y) facing Direction

### ? - help
> Output: Command the robot with:
            L - turn left
            R - turn right
            M - move forward
            ? - this message
            Q - quit

### Q - quit
> Output: Robot shutting down.
